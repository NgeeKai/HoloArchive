Holoarchive — hololive OFFICIAL CARD GAME Card Database

Holoarchive is a fan-made searchable database for the hololive OFFICIAL CARD GAME (hOCG). Browse and search every card across all sets including Booster Packs and Start Decks. Filter by card type (Oshi, Holomem, Support, Cheer), color (White, Green, Red, Blue, Purple, Yellow), rarity (C, U, R, RR, OSR, SR, SEC, UR), bloom level (Debut, 1st, 2nd, Spot), and set (hBP01, hBP02, hBP03, hBP04, hSD01, hSD02, hSD03, hSD04, hSD05, hSD06, hSD07).

Card Sets

  • hBP01 — Booster Pack: Blooming Radiance — The first booster set. Features holomem from hololive JP, EN, and ID generations.
  • hBP02 — Booster Pack: Quintet Spectrum — Second booster expansion with new Oshi cards and Bloom mechanics.
  • hBP03 — Booster Pack: Elite Spark — Third booster set released December 2025.
  • hBP04 — Booster Pack: Curious Universe — Latest booster set released February 2026.
  • hSD01 — Start Deck: Tokino Sora & AZKi — Blue starter deck featuring 0th generation members.
  • hSD02 — Start Deck: Red Nakiri Ayame — Red starter deck.
  • hSD03 — Start Deck: Blue Nekomata Okayu — Blue starter deck featuring Gamers generation.
  • hSD04 — Start Deck: Purple Yuzuki Choco — Purple starter deck.
  • hSD05 — Start Deck: White Todoroki — White starter deck released December 2025.
  • hSD06 — Start Deck: Green Kazama Iroha — Green starter deck released December 2025.
  • hSD07 — Start Deck: Yellow Shiranui Flare — Yellow starter deck released December 2025.

Card Types

  • Oshi — Your main hololive member card. Each player has one Oshi with unique Oshi Skills and SP Oshi Skills.
  • Holomem — Member cards you play on the field. Have HP, Arts attacks, and optional Bloom/Collab effects.
  • Buzz Holomem — Special variant Holomem cards with enhanced abilities.
  • Support — One-time use cards including Items, Tools, Mascots, Events, Fans, and Staff.
  • Cheer — Energy cards attached to your holomem to power their Arts.

About hololive OFFICIAL CARD GAME

The hololive OFFICIAL CARD GAME (hOCG) is a trading card game published by COVER Corp. featuring members of hololive production. Cards are available in Japanese and English editions. All card data and images are copyright COVER Corp. This is an unofficial fan database.
